## 🔧 常用函数速查

```excel
# 预测 / 误差
=FORECAST(新x, 已知y, 已知x)
=TREND(已知y, 已知x, 新x)
=SUMXMY2(观测值, 预测值) / COUNT(预测值)   # MSE

# 最优参数定位
=INDEX(N列, MATCH(MIN(MSE列), MSE列, 0))

# 财务函数
=NPV(贴现率, 第1年起各期现金流) + 第0年金额
=IRR(含第0年的全部现金流)
=PMT(月利率, 期数, -贷款额)

# 规划优化
=SUMPRODUCT(array1, array2)

# 概率统计
=COUNTIF(试验区, ">目标值") / 试验次数
=NORMINV(RAND(), 均值, 标准差)
```
