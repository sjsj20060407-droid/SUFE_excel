## 第二章 数据获取与数据库查询

### 1. 数据获取

| 数据类型 | 操作路径 |
|----------|----------|
| 文本数据 | 数据 → 获取外部数据 → 自文本/自网站 |
| Access数据库 | 自其他源 → Microsoft Query → MS Access Database |
| Excel文件 | 自其他源 → Microsoft Query → Excel Files（没有表：选项→勾选"表"） |

### 2. 数据库查询（Microsoft Query）

**多表查询要点：**
- 确定各字段来源于哪些表，找表间联系（直接/间接/人工建立）
- 选满足条件的最小表集合
- **内连接（重点）**：挑出多表中符合条件的记录
- 双击连接线 → 可改为外连接

**条件设置：**
- 不同列同行 = AND（与）；不同行 = OR（或）
- 注意日期格式：如1996年 → `>=1996-01-01` 且 `<=1996-12-31`；"介于"用 ≥ 和 ≤

### 3. 计算与汇总查询

**计算字段（双击列）：**
```
销售金额 = 订单明细.单价 * 数量 * (1-折扣)
保留两位小数 = Round(订单明细.单价 * 数量 * (1-折扣), 2)
合成姓名 = 员工.姓氏 & 员工.名字   ← Excel只能用&，不能用+
```

**汇总查询：**
- 分类字段：用于分组的字段（列表中除汇总字段外都是分类字段）
- 汇总字段：求和/平均值/计数/最大值/最小值
- 需汇总两类时用 SQL：`SELECT TOP 10...`，`SUM(...) AS ...`，`AVG(...) AS ...`

> ⚠️ 计算每份订单平均销售额：先对订单金额汇总，再求平均值（不能直接平均）

## 第五章 时间序列预测模型

### 无趋势模型

R² < 0.5：认为无趋势

#### 移动平均模型
$$F_{t+1} = \frac{1}{N}\sum_{i=1}^{N}Y_{t-i+1}$$

```excel
预测值 = IF(月份<=N, "", AVERAGE(OFFSET(月份, -N, 1, N, 1)))
MSE = SUMXMY2(观测值, 预测值)/COUNT(预测值)
最优N = INDEX(N取值, MATCH(MIN(MSE表), MSE表, 0))
```

#### 指数平滑模型
$$F_{t+1} = \alpha Y_t + (1-\alpha)F_t$$

- 第2时刻预测值 = 第1时刻观测值
- α ∈ [0,1]，控件步长=1，α'=α/10
- 同移动平均，找最优α

### 趋势预测模型

#### 线性趋势 F = a + bX

| 方法 | 操作 |
|------|------|
| 散点图趋势线 | 添加趋势线+公式+R² |
| 内建函数 | `INTERCEPT(y,x)`/`SLOPE(y,x)`/`LINEST(y,x)` |
| 回归分析 | 数据→数据分析→回归（只适用线性） |
| 规划求解 | 最小化MSE，可变单元格a、b |
| 预测 | `FORECAST(x, 已知y, 已知x)` / `TREND(已知y, 已知x, 新x)` |

> 回归主看**自变量p值**，p<0.05通过检验；残差图在x=0上下均匀=模型好

#### 季节指数模型 $Y_t = S_t \times T_t$

1. 按季度做图
2. 季节指数 = 各季度平均值 / 所有季度平均值（季节指数之和=季节数）
3. 趋势值 $T_t = Y_t / S_t$（按季度一组，4个一组 Ctrl+Shift+Enter，绝对引用 $S_t$）
4. `FORECAST(x, 趋势值, 序号)` → 预测趋势值
5. 预测值 = 预测趋势值 × 季节指数

## 第六章 回归分析预测模型

### 线性回归

**一元：** `Y = a + bX` → 数据分析→回归 / 规划求解

**多元（m≥3）：** 看 adjust_R²，找最优子集
- 最优子集法：所有子模型中 adjust_R² 最高
- 向前增选法：先选一元R²最大，依次加变量直到 adjust_R² 不再提升

### 非线性回归（常考）

**给定模型形式时：**

| 方法 | 说明 |
|------|------|
| 多次规划求解 | 赋初始值→最小化MSE→反复迭代直到MSE不变 |
| 变换法 | 转换为线性，用 INTERCEPT/SLOPE 求参数 |

**变换法常见转换：**

| 模型 | 变换 | 增加列 |
|------|------|--------|
| 幂函数 $y=ax^b$ | 取 $\ln y$, $\ln x$ | 两列 |
| 指数函数 $y=ae^{bx}$ | 取 $\ln y$ | 一列 |
| 对数函数 $y=a+b\ln x$ | 取 $\ln x$ | 一列 |
| 双曲线 $y=a+b/x$ | 取 $1/x$ | 一列 |
| 三次多项式 | 增加 $x^2$, $x^3$ | 两列 → 多元线性 |

## 第七章 成本管理决策模型

### 本量利分析

| 指标 | 公式 |
|------|------|
| 销售收入 | R = p × Q |
| 可变成本 | V = v × Q |
| 总成本 | C = F + V |
| 利润 | π = R - C = (p-v)Q - F |
| 边际贡献 | (p-v)Q |
| 边际贡献率 | k = (p-v)/p = 1-v/p |

**盈亏平衡点：** $Q_0 = F/(p-v)$

**计算方法：**
1. 公式法
2. 单变量求解（数据→模拟分析→单变量求解，目标π=0）
3. 查表加内插值：$x = x_1 + (x_2-x_1)(y-y_1)/(y_2-y_1)$

**目标销量（安全边际）：** $Q_t = (F+\pi_t)/(p-v)$；安全边际 $S = Q_t - Q_0$

### 两种方案的成本决策

相对平衡点（成本相等）：$Q = (F_1-F_2)/(v_2-v_1)$

**折扣采购：**
```excel
折扣 = IF(Q>=最大阈值, 对应折扣, IF(Q>=第二阈值, 对应折扣, 基础折扣))
或用 INDEX(折扣列, MATCH(Q, 阈值列, 1))
```

## 第八章 库存管理决策模型

### 需求确定的库存模型

**库存量公式（开始日期为0）：**
```excel
过去周期数 = INT(当前天/订货周期)
本周期已过天数 = 当前天 - 过去周期数 × 订货周期
库存量 = 订货量 - 日需求量 × 本周期已过天数
```

### 经济订货量（EOQ）

$$C = \frac{kD}{Q} + \frac{hQ}{2} + pD$$

最优：$Q^* = \sqrt{\frac{2kD}{h}}$，此时订货成本=储存成本

### 最佳订货周期模拟（重要15'）

**每天模拟（365天）：**
```excel
起始库存 = 上天结束库存 + 上天订货标志 × 订货量
结束库存 = 起始库存 - 日需求量
订货标志 = IF(结束库存 < 日需求量, 1, 0)
订货成本 = 订货标志 × 每次订货成本
储存成本 = 单位储存成本/天 × (起始+结束)/2
```

最优订货周期 = `INDEX(周期列, MATCH(MIN(MSE), 年总成本列, 0))`

### 随机需求库存模型

```excel
日需求量 = RANDBETWEEN(最小, 最大)
订货标志 = IF(结束库存 < 安全库存, 1, 0)
结束库存 = IF(起始 > 日需求, 起始-日需求, 0)
缺货量 = IF(起始 > 日需求, 0, 日需求-起始)
服务水平 = SUM(365天未缺货)/365
```

**100次模拟：** 二维模拟运算表，列=订货量，行=虚拟变量（引用空白单元格）

## 第九章 投资决策分析模型

### 财务函数总结

| 函数 | 功能 | 关键参数 |
|------|------|----------|
| `FV(rate,nper,-pmt,-pv)` | 将来值 | pmt/pv为负 |
| `PV(rate,nper,-pmt,-fv)` | 等额现值 | pmt/fv为负 |
| `NPV(rate,value1,...)` | 不等额现值 | 第0年单独加 |
| `IRR(values, guess)` | 内部收益率 | 含第0年现金流 |
| `PMT(rate,nper,-pv)` | 等额每期还款 | PMT=PPMT+IPMT |
| `PPMT(rate,per,nper,-pv)` | 某期本金 | t↑ 本金↑ |
| `IPMT(rate,per,nper,-pv)` | 某期利息 | t↑ 利息↓ |
| `ISPMT(rate,per-1,nper,-pv)` | 等本金某期利息 | per从0开始 |
| `RATE(nper,pmt,pv)` | 利率 | 迭代计算 |
| `NPER(rate,pmt,pv)` | 还款期数 | |

> ⚠️ **净现值 = NPV(第1年及以后现金流) + 第0年金额**（第0年不折现）

### 两方案净现值相等的贴现率

```excel
方法1：IRR(方案1现金流 - 方案2现金流)
方法2：查表+内插值（模拟运算表：贴现率→净现值差，找正负两点插值）
```

### 蒙特卡洛风险分析框架

1. **输入区**：全部固定参数
2. **生成区**：随机数（正态/均匀/离散/泊松）
3. **输出区**：中间结果 + 最终净现值
4. **试验区**：模拟运算表，x=虚拟变量，y=净现值
5. **统计区**：均值、标准差、最大/最小值
6. **图形区**：概率密度曲线，`COUNTIF(试验表,">目标值")/次数`

## 第十章 最优化决策模拟

### 线性规划（重点）

**规划求解设置：**
- 线性规划/整数约束 → 选"单纯线性规划"
- 非线性 → 选"GRG"

**数组求和：**
```excel
=SUMPRODUCT(array1, array2)  ← 对应元素相乘求和
=SUM(array1 * array2)  ← Ctrl+Shift+Enter
```

### 常见规划问题框架

| 问题类型 | 目标函数 | 决策变量 | 核心约束 |
|----------|----------|----------|----------|
| 产品混合 | 最大化利润 | 生产件数 | 时间/人力 ≤ 上限，件数 ≥ 0 且整数 |
| 选址问题 | 最小化费用 | 是否选该位置（0-1变量bin） | 覆盖次数 ≥ 1 |
| 运输问题 | 最小化运费 | 各供应商→各门店供货量 | 供≤产能，需≥需求量 |
| 采购管理 | 最小化总成本 | 每月采购量 | 月末库存≥安全库存 |
| 资金管理 | 最大化第T年本金利息 | 每年各投资方案金额 | 每年现金余额≥0 |

### 最优化决策模拟模型框架（随机变量情形）

1. 写出最优解表达式（如最优单价公式）
2. 生成随机变量（需求量、成本等）
3. 模拟1000次（二维模拟运算表，虚拟变量做行）
4. 统计均值、标准差等

## 🔑 考试高频注意事项

1. **模拟运算表排序** → 用 `LARGE()`
2. **柱形图x轴平均值线** → x轴必须包含"全部"数据
3. **数据透视表NULL** → 右键→选项→对错误值显示→空格（Enter）
4. **日期"全部"条件** → 设为最早日期到最晚日期范围，而非 `""`
5. **折扣阈值自变量** → 输入 阈值-0.0001 和 阈值 两个点
6. **等本金还款** → `ISPMT(rate, per-1, nper, -pv)`，per从1开始
7. **按月还款** → 用月利率（年利率/12），按月做现金流
8. **半年到期现金流** → `=SUM(现金流/(1+rate)^年份)` 手动折现
9. **净现值** = `NPV(第1年后现金流)` + 第0年金额（第0年不用NPV）
10. **先规划求解，再 Power Pivot**
11. **时间序列图形** → 折线图，x轴为文本类型
